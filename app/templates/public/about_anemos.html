{% if lang == 'en' %}The Anemos label{% else %}Le label Anemos{% endif %}

{% if lang == 'en' %} Anemos is an independent certification that attaches a verified CO₂ score to each maritime voyage. Created specifically for sailing cargo operators, it gives shippers a reliable, third-party-verified measure of the climate impact of their goods. {% else %} Anemos est une certification indépendante qui attribue un score CO₂ vérifié à chaque voyage maritime. Conçu spécifiquement pour les opérateurs de cargo vélique, il offre aux chargeurs une mesure fiable et vérifiée par un tiers de l'impact climatique de leurs marchandises. {% endif %}

{% if lang == 'en' %}Measure{% else %}Mesurer{% endif %}

{% if lang == 'en' %} Actual fuel consumption is monitored continuously on board, voyage by voyage, using instruments certified under EU MRV regulation. {% else %} La consommation réelle est surveillée en continu à bord, voyage par voyage, via des instruments certifiés dans le cadre du règlement EU MRV. {% endif %}

{% if lang == 'en' %}Verify{% else %}Vérifier{% endif %}

{% if lang == 'en' %} An accredited independent auditor reviews the data annually and certifies the emission factor used in all calculations. {% else %} Un organisme indépendant accrédité audite les données chaque année et certifie le facteur d'émission utilisé dans tous les calculs. {% endif %}

{% if lang == 'en' %}Certify{% else %}Certifier{% endif %}

{% if lang == 'en' %} Each shipment receives a named Anemos certificate — a traceable, downloadable PDF ready for your CSR reporting. {% else %} Chaque expédition reçoit un certificat Anemos nominatif — un PDF traçable et téléchargeable, prêt pour vos rapports RSE. {% endif %}
